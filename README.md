# Ícaro de Sousa Rodrigues


Sou estudante de **Engenharia de Software** e desenvolvedor **Full Stack**, com experiência prática na construção, manutenção e evolução de aplicações web e sistemas em produção.

Atuo principalmente com **React, Next.js, TypeScript, Node.js, NestJS e PostgreSQL**, desenvolvendo interfaces, APIs REST, integrações, autenticação, regras de negócio, modelagem de dados, testes automatizados e processos de publicação.

Minha trajetória combina atuação profissional, projetos freelancers, trabalho voluntário e projetos acadêmicos em equipe. Também possuo experiência com **Java, Spring Boot, C#, ASP.NET Core, Python, Docker, MySQL e MongoDB**, sempre buscando aplicar boas práticas, organização de código e soluções fáceis de manter.
